# 50代からITへ挑戦中｜未経験からの学びと記録

はじめまして。現在、IT分野へのキャリアチェンジを目指して学習を進めている50代です。  
職業訓練校でWeb開発やプログラミングに触れ、JavaやMySQLなどの技術を学びながら、ITパスポート・Java資格を取得しました。
